プライバシーポリシー (Privacy Policy)

日本語 (Japanese)

Jun（以下、「当方」といいます。）は、提供するChrome拡張機能「TubeEQ」（以下、「本拡張機能」といいます。）における利用者情報の取り扱いについて、以下のとおりプライバシーポリシー（以下、「本ポリシー」といいます。）を定めます。

1. 収集する情報と利用目的

本拡張機能は、その機能を提供・向上させるために、以下の情報を限定的な目的で利用します。

ウェブサイトのコンテンツ（音声データ）
YouTubeページ上のビデオプレーヤーから音声ストリームを取得します。これは、イコライザー効果を適用するという本拡張機能の中核的な機能のためにのみ使用されます。音声データが外部のサーバーに送信されたり、保存されたりすることはありません。

ウェブ履歴（閲覧中のページのURL）
ユーザーが現在閲覧しているウェブページのURL情報を一時的に利用します。これは、本拡張機能がYouTubeのウェブサイト上でのみ動作するように、現在地が対象ページであるかを判断するためにのみ使用されます。閲覧履歴が保存されたり、外部に送信されたりすることはありません。

ユーザーのアクティビティ（操作情報）
本拡張機能のポップアップUI内でのクリック、ドラッグ、キーボード操作などの情報を利用します。これは、ユーザーによるイコライザー設定の変更や機能の制御を可能にするためにのみ使用されます。これらの操作情報がユーザーの行動を追跡する目的で記録されたり、外部に送信されたりすることはありません。

設定情報
ユーザーが設定したイコライザーのプリセットや各種設定は、ユーザーの利便性のために、chrome.storage.local APIを用いてブラウザ内にのみ保存されます。この情報が当方のサーバー等に送信されることはありません。

2. 情報の第三者への提供

本拡張機能が取り扱うすべての情報は、利用者のコンピュータ（ブラウザ内）で処理・保存が完結します。当方を含む第三者のサーバーへこれらの情報が自動的に送信・提供されることは一切ありません。

3. お問い合わせ

本ポリシーに関するご不明点などがございましたら、以下の連絡先までお問い合わせください。
連絡先: peq.youtube@outlook.com

4. プライバシーポリシーの変更

当方は、法令の変更等に伴い、本ポリシーを事前の予告なく変更することがあります。

English

Jun ("the developer") establishes the following privacy policy ("this Policy") regarding the handling of user information in the Chrome extension "TubeEQ" ("this Extension").

1. Information We Collect and How We Use It

To provide and improve its functionality, this Extension uses the following information for limited purposes:

Website Content (Audio Data)
We access the audio stream from the video player on YouTube pages. This is done solely to apply equalizer effects, which is the core function of this Extension. Audio data is never sent to or saved on external servers.

Web History (Current Page URL)
We temporarily use the URL of the web page you are currently viewing. This is done only to determine if you are on a YouTube website, ensuring that the Extension functions exclusively on the intended site. Your browsing history is never saved or transmitted externally.

User Activity (Operational Information)
We use information such as clicks, drags, and keyboard inputs within the Extension's popup UI. This is used only to allow you to change equalizer settings and control the Extension's features. This operational information is not recorded to track user behavior and is never sent externally.

Settings Information
Your equalizer presets and various settings are saved only within your browser using the chrome.storage.local API for your convenience. This information is never transmitted to our servers or any other third party.

2. Sharing of Information with Third Parties

All information handled by this Extension is processed and stored entirely on the user's computer (within the browser). This information is never automatically transmitted or provided to any third-party servers, including those of the developer.

3. Contact Us

If you have any questions about this Policy, you may contact us at:
Contact: peq.youtube@outlook.com

4. Changes to This Privacy Policy

The developer may amend this Policy without prior notice in response to changes in laws and regulations or for other reasonable causes.
